# Formulário de Cadastro

Este repositório contém o código HTML para um formulário de cadastro. O formulário é dividido em duas seções principais: dados pessoais e endereço.

## Tecnologias Utilizadas

- HTML5

- CSS3

## Visualização
Para visualizar o formulário, você pode clonar o repositório e abrir o arquivo `index.html` em qualquer navegador moderno. 


## Estrutura do Formulário

O formulário inclui os seguintes campos:

- Dados Pessoais (Nome, Idade, Telefone, E-mail)

- Endereço (CEP, Estado, Cidade, Bairro, Endereço, Número, Sem número, Complemento)
